核心功能

🔑

BYOK Profile

以 YAML 管理 API 端點、金鑰與模型清單，金鑰以 OS keychain 加密儲存。

🛡️

零環境污染

環境變數僅注入子程序，Shell 與系統設定檔永不被修改。

🧩

多工具支援

Copilot、Codex、Codex App、Claude、Pi，一個 byok 全部搞定。

📦

自我更新

byok update 自動檢查 GitHub Releases 並原子替換執行檔。

🖥️

互動模型選單

多模型 Profile 在終端機顯示上下鍵互動選單，直覺切換。

🔒

Keychain 加密

API 金鑰以 OS keychain（macOS / Windows / Linux）加密儲存。

三分鐘上手

兩步驟，馬上開始使用自己的 API Key

Step 1

設定 Profile

建立一個 BYOK profile，指定 provider、API 端點與金鑰。金鑰預設存入 OS keychain 加密儲存。

byok config add my-profile
Step 2

啟動工具

選擇你要使用的 CLI 工具，一行指令啟動。環境變數僅注入子程序，父程序零污染。

Copilot CLI

byok launch copilot

Codex CLI

byok launch codex

Codex App

byok launch codex-app

Claude

byok launch claude

Pi

byok launch pi

安裝方式

無須依賴 — 下載即用，單一執行檔

✅ 無需 Node.js、無需 Python、無需任何執行階段 — 下載單一執行檔即可使用
curl -L https://github.com/IISI-2209026/LlmByok/releases/latest/download/byok-latest-linux-amd64 -o /usr/local/bin/byok && chmod +x /usr/local/bin/byok
curl -L https://github.com/IISI-2209026/LlmByok/releases/latest/download/byok-latest-darwin-amd64 -o /usr/local/bin/byok && chmod +x /usr/local/bin/byok
Invoke-WebRequest -Uri "https://github.com/IISI-2209026/LlmByok/releases/latest/download/byok-latest-windows-amd64.exe" -OutFile "$env:LOCALAPPDATA\byok.exe"
